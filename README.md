# tickmyday-java-ee-todolists-web-app
 A simple todo lists web app built using java EE

## Java EE technologies used
* Servlets
* JSP
* JSTL
* JDBC

## Other technologies used
* Bootstrap
* Fetch API (for AJAX calls with no body)
* MySql database
